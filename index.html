Introduction

Computer hardwares breaks into different computer components that has to be combined together to produce a working computer. This includes CPU ( Central Processing Unit ), Motherboard, GPU ( Graphic Processing Unit ), RAM ( Random Access Memory ), Storage Disks and a PSU ( Power Supply Unit ) all the hardwares above combined to produce a working computer.
And all the hardwares above does different jobs inside a running computer unit, and all of them are unique. Therefore if any of the hardware components above are missing a computer would not run normally.
